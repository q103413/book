### 初始化

### 第一步：进入 Hive 根目录（必须！）

```
cd /usr/local/apache-hive-3.1.3-bin
```

### 第二步：强制删除残留的脏数据（核心！）

```
rm -rf metastore_db derby.log
```

### 第三步：重新初始化 Derby（这次 100% 成功）

```
schematool -dbType derby -initSchema
```
